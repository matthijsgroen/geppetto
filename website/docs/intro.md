# Getting started

Hey! Welcome to Geppetto! With Geppetto you can create 2D animations from your image.

## Goals

1. Free to use & distribute
2. Open source
3. Capable of animations & runtime controls
4. Energy efficient
5. Made for the web

## Features

- Animation
- Live controls
- Color effects
- Small in size
